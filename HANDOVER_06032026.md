## HOW TO UPLOAD TO GITHUB (step by step)

### Method — Edit directly on GitHub (no download/rename needed):
1. Go to the repo URL (e.g. github.com/avshi2-maker/work-journal)
2. Click `index.html` in the file list
3. Click the **pencil icon ✏️** (top right of file view)
4. Press **Ctrl+A** → select all existing content → **Delete**
5. Open the new file from Claude → **Ctrl+A** → **Ctrl+C**
6. Click inside the GitHub editor → **Ctrl+V** (paste)
7. Scroll down → click **Commit changes** → **Commit changes** again
8. Wait 2 minutes → GitHub Pages rebuilds automatically

### If GitHub Pages shows "Queued" for more than 5 minutes:
1. Settings → Pages → Source → change to **None** → Save
2. Wait 10 seconds
3. Change back to **main / (root)** → Save
4. Wait 2 minutes → should go green ✅
